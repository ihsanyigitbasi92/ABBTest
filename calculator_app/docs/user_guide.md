# User Guide

## Introduction
Welcome to the Calculator Application. This application allows you to perform basic arithmetic operations, scientific calculations, statistical analysis, and matrix operations. It also features a graphical user interface for ease of use.

## Features
- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- Scientific calculations: Trigonometric functions, Logarithmic functions, Exponential functions
- Statistical calculations: Mean, Median, Mode, Standard deviation
- Matrix operations: Addition, Subtraction, Multiplication, Determinant
- Graphical User Interface (GUI) with interactive buttons and display panel

## How to Use
1. Launch the application by running `main.py`.
2. Use the buttons on the GUI to input your calculations.
3. Press '=' to evaluate the expression.
4. View the result in the display panel.
5. For matrix operations, input matrices in the format `[[a, b], [c, d]]`.

## Error Handling
- Division by zero will result in an error message.
- Invalid input types will result in an error message.
